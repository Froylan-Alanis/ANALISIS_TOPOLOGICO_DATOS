Análisis de Localidades (ITER)

Censo de Población y Vivienda 2020

Sistema de consulta y análisis topológico de datos estadísticos

Implementación en Flask + SQLite + Bootstrap + TDA

Tema de investigación

El proyecto desarrolla un sistema de análisis topológico de datos (TDA) utilizando información del Censo de Población y Vivienda 2020 del INEGI, específicamente el archivo ITER de localidades.

Mediante herramientas estadísticas y métricas espaciales, el sistema permite explorar patrones poblacionales, similitud entre localidades, agrupamientos demográficos y detección de anomalías utilizando técnicas de análisis multidimensional.

Justificación

Se eligió esta temática debido a la importancia del análisis poblacional y territorial en la comprensión de fenómenos demográficos y sociales. El uso de TDA permite representar las localidades como espacios métricos y estructuras topológicas, facilitando la identificación de relaciones, similitudes y comportamientos atípicos dentro del conjunto de datos censales.

Descripción general

Este sistema analiza información del Censo de Población y Vivienda 2020 utilizando técnicas de Análisis Topológico de Datos (TDA).

Variables principales
  • NOM_ENT: Nombre de la entidad federativa
  • NOM_MUN: Nombre del municipio
  • NOM_LOC: Nombre de la localidad
  • POBTOT: Población total
  • POB0_14: Población de 0 a 14 años
  • POB15_64: Población de 15 a 64 años
  • POB65_MAS: Población de 65 años y más
  • LATITUD / LONGITUD: Coordenadas geográficas
Conceptos TDA utilizados
  • Nubes de puntos
  • Espacios métricos
  • Distancia Manhattan
  • Detección de outliers
  • Agrupamiento topológico
